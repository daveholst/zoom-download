Install youtube-dl cli tool

File Name

Enter the filename you would like with a .mp4 suffix


How to get File URL

Goto the zoom video replay with the network panel of dev tools open and search by mp4 to get the network request for the video. In this example it is the top request with Content-Type: video/mp4.



On the right side of the dev tools window/pane copy the request URL and paste into the File URL box.


How to get Cookie

Below the Request URL, go down to Request Header and copy the cookie and paste into the Cookie Header Box

Hit Generate and paste the command into your terminal!
